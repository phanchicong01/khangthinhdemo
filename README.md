# Khang Thịnh Investment - Website

Website giới thiệu công ty Khang Thịnh Investment.

## Tech Stack

- Next.js 15 (App Router, Static Export)
- Tailwind CSS 4
- TypeScript

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
# Output: /out/ (static files)
```

## Deploy

Static export → có thể deploy lên GitHub Pages, Netlify, Vercel.
